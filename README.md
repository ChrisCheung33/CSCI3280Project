# CSCI3280 Project

- This is the project of CSCI3280 Group 13 in the 2022-2023 2nd semester

## Start the project
- make sure you are in the directory `Music_Player`
    ```bash
    cd Music_Player
    ```

- install modules
    ```bash
    pip install -r requirements.txt 
    ```

- run the project
    ```bash
    python main.py
    ```
